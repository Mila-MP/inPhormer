Step 1  
Make sure poetry is installed  
``pipx install poetry``  
``poetry --version``

Step 2  
clone repo on your local machine  

Step 3  
Install the poetry environment  
``poetry install``  

Step 4
Activate poetry environment
``poetry env activate``
Copy the output link to your command line and enter. This should activate the poetry environment.

Step 5   
Install pytorch [here](https://pytorch.org/get-started/locally/)
